# Open WebUI + LiteLLM

## Stack

- Open WebUI - Chat interface
- LiteLLM - OpenAI proxy with tracking
- SearXNG - Privacy-respecting metasearch engine
- PostgreSQL 17 - Database
- Caddy 2 - Auto HTTPS

## Quick Start

```bash
cp .env.example .env
# Edit .env with your values
docker compose up -d
```
